In this program, you can
(1)create a new maze(the number of rows and columns must be >=5)
(2)read a maze from file 

the program will show the original maze in numbers:

1 - wall
0 - path 
200 - start point
201 - destination
202+ - treasure

One suggested solution will be printed automatically. 
The first step will be shown and then you will have to press ENTER to see the next step.
The number of treasures found for this solution will be printed at the end of the program.

NOTICE!
Please avoid illegal input as it might result in the breakdown of the program.
